Using Obsidian for rhizomic research

Abstract

This thesis is written in Obsidian, as a collection of wikilinked Markdown notes. In order to render each file as a Web page, with support for citations and wikilinks, it was necessary to construct a software script which could automate the various transformations required. This document explains the steps involved in getting this to work …. almost as if it’s a free standing article!
